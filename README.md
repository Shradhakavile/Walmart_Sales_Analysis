# Walmart_Sales_Analysis

**Project Overview**
This project is an end-to-end data analytics workflow designed to analyze Walmart sales performance and generate business insights using SQL, Python, and Power BI.

The goal of this project is to demonstrate real-world data analysis skills including:
Data storage and querying
Data cleaning and transformation
Exploratory data analysis
Business insight generation
Interactive dashboard development
This project simulates how a data analyst works with raw transactional data to produce meaningful insights for decision-making.

**Tools Used**
**SQL (MySQL)**
Database creation
Table schema design
Data import and validation
Aggregation queries
Business metrics calculation

**Python**
Data loading and inspection
Data type corrections
Time-based feature engineering
Exploratory Data Analysis (EDA)
Trend visualization

**Power BI**
Interactive dashboard design
KPI tracking
Data filtering using slicers
Business-focused visual storytelling

**Dataset Description**
The dataset contains Walmart sales transactions with 1000+ records and the following attributes:
Invoice ID, Branch, City, Customer Type, Gender, Product Line, Unit Price, Quantity, Tax, Total Sales, Date & Time, Payment Method, Cost of Goods Sold (COGS), Gross Income, Gross Margin Percentage, Customer Rating

**Project Workflow**
**1. Data Storage (SQL)**
Created database and sales table
Imported transaction dataset
Verified row counts and structure
Calculated total revenue and other summary metrics

**2. Data Analysis (Python)**
Loaded dataset using Pandas
Converted time column to hourly format
Generated hourly sales trends
Performed basic statistical exploration

**3. Business Dashboard (Power BI)**
Created an interactive dashboard to monitor:
Total Revenue
Total Gross Income
Total Transactions
Average Customer Rating
Sales by Branch
Revenue by Product Line
Customer Payment Preference

The dashboard allows filtering by:
Branch
Customer type
City

**Key Insights**
Branch C contributes the highest share of total revenue.
Sales are relatively consistent across major product categories.
Digital payment methods represent a large share of transactions.
Customer ratings remain consistently high across branches.
